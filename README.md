# python-ui-template
This is a template I have generated to use on some projects. It represent a main window, a sidebar to display information and a log section. The code is intentionally kept as a single large file so it can easily copied and pasted to various AI tools.

As a starting point, I would suggest to load this into an AI to get rid of the comments, or better yet replace them with single line short ones and then split into the structure you have in mind.
